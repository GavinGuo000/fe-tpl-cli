## 依赖版本
Nodejs > 8.9 && npm >5.6

## 模板介绍

#### react-hooks-tpl
* 支持React hooks，react版本需16.8及以上
*  支持react-redux7.1，项目内使用 useSelector, useDispatch, useStore，不再使用 connect

#### react-official-tpl
* 官方早期推荐的版本，也是现有大部分react项目的技术栈

## 安装
``` npm i fe-tpl-cli -g ```

## 创建项目
``` fe-tpl-cli init ```
